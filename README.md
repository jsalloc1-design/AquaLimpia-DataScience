# Proyecto AquaLimpia - Ciencia de Datos

## Objetivo
Analizar el desempeño de las plantas de tratamiento mediante herramientas de ciencia de datos.

## Herramientas utilizadas
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Plotly
- SciPy
- Joblib

## Proceso realizado
1. Carga del dataset.
2. Limpieza y validación de datos.
3. Análisis exploratorio.
4. Cálculo de eficiencia.
5. Dashboard exploratorio.
6. Exportación de resultados.

## Resultados
- Evaluación de eficiencia.
- Identificación de incumplimientos.
- Dashboard interactivo.
- Reportes para operaciones y gestión ambiental.
